ValCarBid Project - Version 2.0

Overview
--------
ValCarBid is a robust platform designed to streamline the car bidding process, ensuring a user-friendly experience for both buyers and sellers. In this version, we focused on enhancing functionality and addressing minor bugs to improve overall user satisfaction.

Updates in Version 2.0
-----------------------
Key Improvements:
- Bug Fixes: Resolved various minor bugs to enhance system stability and performance.

- Enhanced Account Deletion Logic:
  * Rewrote the logic for handling user account deletions.
  * Implemented warning notifications for users attempting to delete their accounts without eligibility, ensuring clearer communication and a smoother user experience.
